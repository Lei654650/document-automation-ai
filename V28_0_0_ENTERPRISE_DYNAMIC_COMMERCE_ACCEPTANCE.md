# V28.0.0 版本变更报告

## 新增
- 首页动态处理演示：进度、阶段、页数、预计剩余时间、实时日志。
- 五档套餐、支付流程、套餐对比、FAQ 与安全信任说明。
- 支付成功后自动生成 License，且重复 Webhook 不重复发放。
- License 查询与设备绑定接口。
- 企业验收中心，可自动执行关键数据库、支付、钱包、License 与插件检查。

## 验收入口
顶部导航点击“企业验收”，或访问前端后进入 Acceptance 页面。

## 支付说明
真实收款仍需在环境变量中配置 Paddle/PayPal/Stripe 密钥。测试模式用于本地验收订单、到账与授权流程。
