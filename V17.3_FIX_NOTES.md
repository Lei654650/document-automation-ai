# V17.3.0 Stability Hotfix

- Large Excel files (>= 8 MB) use read-only streaming preview during order creation.
- Full workbook loading is deferred to processing, preventing upload UI stalls.
- Backend stdout/stderr and tracebacks are appended to logs/backend.log.
- Start_All reuses an existing backend on port 8000 instead of starting duplicates.
- Verified with a 49.4 MB XLSX: order API returned HTTP 200 in under one second in local test.
