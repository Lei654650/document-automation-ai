# V30.1.5 Task Lifecycle Refinement

- Removed archive actions and archive filters from Live Processing Center.
- Added retry and permanent deletion actions with explicit irreversible confirmation.
- Permanent deletion removes owned task records, processing logs, database file rows, and stored source/delivery files.
- Added real start/completion timestamps, duration, estimated AI credits, processor, provider, and model fields.
- Delivery area now labels actual generated formats and only lists files that exist in the task record.
- Project Center archive behavior remains unchanged for long-term enterprise project management.
