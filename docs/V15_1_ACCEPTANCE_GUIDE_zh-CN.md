# V15.1 复杂验收指南

测试目录：`samples/v15_1_acceptance`

建议顺序：
1. Word：多页合同、FAT、SOP、三语手册。
2. PDF：文字型多页 PDF，分别测试 Word/PDF 输出。
3. Excel：公式、数据验证、条件格式、图表。
4. PPT：8页演示文稿，检查翻译和原格式保留。
5. Image：清晰发票、模糊倾斜发票、铭牌、工艺卡。
6. ScanPDF：扫描发票和工艺卡，必须开启 OCR。
7. CSV：500行追溯数据。
8. ZIP：混合文档批量和嵌套 ZIP。

每项检查：上传、处理完成、输出扩展名、可打开、页/表/幻灯片数量、公式/图片/表格、翻译完整性、批量隔离。
