# Document Automation AI V16.3.0 验收集中修复报告

本版本基于 V16.2.0 的人工企业验收结果集中修复，不包含压力测试；压力测试安排在本版本功能回归全部通过后进行。

## 已修复

- 结果页文件明细默认折叠，避免大批量任务页面无限增长。
- 交付结果按成功/失败分区，成功文件支持搜索、每页 20 条分页。
- 状态页持续显示本次上传文件数量和文件名摘要。
- 下载弹窗区分“用户选择的浏览器保存位置”和“项目输出目录”。
- 图片及文件名明确标识为扫描件的 PDF 自动启用 OCR；不再自动启用翻译。
- PDF/扫描 PDF 选择“保留原格式”时生成 PDF，不再错误交付未勾选的 DOCX。
- 中英/中越双语目标向翻译引擎传递明确双语输出要求。
- 部分失败时，成功文件与失败原因分别展示，成功文件仍可下载。
- 版本统一更新到 V16.3.0。

## 验证

- 前端 Vite 生产 Build：通过（1567 modules）。
- 后端 pytest：12/12 通过。
- Python compileall：通过。
- PDF 原格式规则专项测试：PDF + XLSX 生成成功，未生成未勾选 DOCX。
