# V18.0.2 Excel 批量翻译性能修复

## 修复内容

- Excel 翻译从逐单元格 API 请求改为批量请求。
- 每批默认最多 60 条、7000 字符，可通过环境变量调整。
- 自动跳过空白、数字、日期样式文本、公式、网址和纯代码标识。
- 相同文本通过缓存只翻译一次。
- 只遍历工作簿中实际存在的单元格，避免格式扩展到 XFD 列时扫描数千万个空单元格。
- 处理页面显示批次数、处理数量、跳过数量和预计剩余时间。
- 批量 JSON 响应异常时自动降级，保证输出完整性。

## 验证

- 后端自动测试：15 项通过。
- st02 客户文件扫描：5385 个需翻译单元格，10540 个单元格被安全跳过。
- 模拟批量翻译：90 次批量请求完成，替代原来的 5385 次单元格请求。
- 前端生产构建通过。
- 后端健康接口启动检查通过。
